Release Notes for 1.0.10 (stable)

The latest stable version of OpenSLP is 1.0.11.  OpenSLP-1.0.11 contains a few minor bug fixes that some OpenSLP users will be interested in.  See ChangeLog in tarball for details.

Those that are investigating OpenSLP as part of a development or porting project should look at the 1.1.x development releases.  This is where the development action will be.  Those that are deploying OpenSLP in production systems should continue to look at OpenSLP 1.0.11
until the 1.1.x code becomes stable enought for production use.

Release Notes for 1.1.3 (development)

OpenSLP 1.1.x are a develper releases of the cutting edge OpenSLP code.  If you are looking for a stable release, you should download the latest revision of the 1.0.x series (above):
  Changes:
  • The main new feature is the very good DHCP work that has been contributed jcalcote@novell.com
  • Preliminary integration of new filter and predicate code based on work done by Mike Day. Code should be operational by next relea
Known Issues:
  • Much of the new code is not ported to platforms other than Linux.  Please feel free to send patches for your OS directly to me or to the openslp-devel@sourceforge.net mailing list.
