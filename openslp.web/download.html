New in 0.8.1

  • Backwards compatible SLPv1 Passive DAAdvert support
  • Significant 64-bit cleanup work
  • Bugs related to unreturned errorcodes (-19 was being returned for almost everything)
  • autoconf/automake changes to do things "The Right Way"
  • SLP_MAX_LIFETIME services should now age and refreshed correctly
  • Library code hardened more against bad DAs
  • Code changes to the way "my host and domain hame" is determined
  • Changes as suggested by a Security Audit from Olaf Kirch
