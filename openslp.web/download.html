README 0.7.1

As far as enhancements go OpenSLP release 0.7.1 just slightly more stable release of 0.7.0. Several bug fixes (that kept 0.7.0 from working in some cases) have been added.  For developers,  OpenSLP 0.7.1 introduces a new make file system that uses automake/autoconf...  one step closer to our goal of makeing OpenSLP easier to port and maintain.
 

CHANGELOG 0.7.1

  • Fixed 109269 - SLPFindSrvs callback no longer gives a LAST_CALL
  • Fixed a short coming of SLPEscape, it did not handle CTL, which it now does
  • Removed all old Makefile and install scripts and replaced them with Automake/Autoconf ones
