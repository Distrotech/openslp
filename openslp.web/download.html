Release Notes for 1.0.0

OpenSLP 1.0.0 is the first "stable/complete" release of OpenSLP.  It is highly recommended that all OpenSLP users upgrade to this release.
 
The following is a shortened list changes for 1.0.0:
  • Fixes all known bugs (see SourceForge bug tracker http://sourceforge.net/tracker/?group_id=1730&atid=101730 for details)
  • Duplicate results in sync API calls are now removed
  • Fixed several SEGVs attributed to parsing bugs
  • Fix "echo to ourselves" bug when slpd is running as a DA
  • Cosmetic logging changes
  • ALL PLANNED FEATURES ARE SUPPORTED IN THIS RELEASE
