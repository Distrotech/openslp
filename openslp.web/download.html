Release Notes for 1.0.5

OpenSLP-1.0.5 fixes some important bugs. It is highly recommended that all OpenSLP users upgrade to this release.

The following is a shortened list changes for 1.0.5:

  • Minor changes to the way that settings from /etc/slp.conf affect active and passive discovery.
  • Applied fix that makes it so that errorcodes for service-agent and directory-agent SrvRqsts are returned correctly.
  • Fixed possible SEGV when reconnect to DA failed.  With out this change it would be possible to need a NULL socket structure to the outgoing socket list.  This would be very bad. Properly rooted startup scripts
  • Changed declaration and definition for strncasecmp and strcasecmp so that there is only one place for them in slp_compare.c and slp_compare.h
  • Honor scopes in SrvRqsts for the special "service:directory-agent" type
  • Fixed problem with SLPEscape() as patched by Timothy Zurcher
